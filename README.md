Put Skins In /UserData/Skins/(insert object name)/(insert texture type)

Example: /UserData/Skins/Pillar/Main.png will texture the pillar

# Object Names:
Pillar

Cube

Ball

Disc

Wall

Player/Suit

Player/ShiftStone


# Texture Types:
Main.png

Normal.png

Mat.png

Grounded.png

For Shift Stones Add _L or _R To The Texture Name
